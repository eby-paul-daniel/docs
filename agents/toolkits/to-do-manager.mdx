# To-Do Manager Toolkit

Provides persistent memory for tracking tasks across conversations. The toolkit maintains a structured task database that persists across sessions.

![To-Do Manager configuration with features and Update Todos tool](/img/agents-toolkits/21.jpg)

## Available Tools (1/1)

- **Update Todos**: Create/update/delete todos via JSON. Current todo list auto-displays after any operation. Supports merge and replace modes. No explicit limit.

## Use Cases

- **Project Management**: Track multi-step projects with tasks.
- **Customer Onboarding**: Manage onboarding checklists.
- **Content Production**: Coordinate content creation workflows.
- **Issue Tracking**: Maintain lists of bugs and feature requests.
- **Personal Productivity**: Manage personal to-do lists.
