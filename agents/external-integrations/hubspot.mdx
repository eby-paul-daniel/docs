# HubSpot

## Configuration

![HubSpot configuration with default contact properties and tool selection](/img/agents-toolkits/35.jpg)

- **Default Contact Properties**: firstname, lastname, email, company
- **Authentication**: OAuth
- **Access**: Comprehensive CRM capabilities including contacts, companies, deals, and tickets

## Available Tools (4/4)

- **Search Objects**: Full-text search across contacts, companies, and deals.
- **Query Objects**: Query objects with filters and conditions.
- **Manage Objects**: Create, update, or delete CRM records.
- **Get Properties**: List all properties for object types.

## Use Cases

- **Marketing Automation**: Sync leads from web forms and events into HubSpot.
- **Sales Pipeline Management**: Track deals through stages and forecast revenue.
- **Lead Enrichment**: Enhance contact records with external data.
- **Customer Support**: Auto-create tickets from emails or chat.
- **Data Synchronization**: Keep HubSpot in sync with other business systems.
